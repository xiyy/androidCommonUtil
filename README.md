# androidCommonUtil
Android通用工具类插件，引用该工具类插件：  
1、在project的build.gradle中添加   
<pre name="code" class="java">
allprojects {
    repositories {
        ......
        maven { url 'https://jitpack.io' }
    }
}
</pre>
2、在module的build.gradle中添加  
<pre name="code" class="java">
implementation 'com.github.xiyy:androidCommonUtil:tag'
</pre>
